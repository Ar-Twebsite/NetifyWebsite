What to know before you start

Can anyone start using an API?

No. This page covers technical projects assessed and defined case by case.

What do you need to assess one?

A precise use case, example material, expected results, responsibilities, and technical expectations.

What is decided on the first call?

Whether the case deserves deeper technical work. A ready-made integration is not promised.
